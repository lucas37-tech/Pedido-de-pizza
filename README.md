# Pedido-de-pizza

Project developed during a Web Development class as part of my undergraduate studies in the Systems Analysis and Development program.

## Technologies

- HTML5

## Features

- Customer name input
- Email input
- Pizza size selection
- Pizza flavor selection
- Additional toppings selection (Borda and Bacon)
- Delivery date selection
- Comment field for special instructions
- Form submission using the GET method

## Learning Objectives

- Learn the fundamentals of HTML forms
- Practice using different HTML input types
- Understand the difference between GET and POST request methods
- Build a simple form for collecting user input
- Organize form elements using standard HTML tags
